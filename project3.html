Konteks & Tujuan

Platform makanan online menghadapi tantangan memahami perilaku pelanggan yang beragam — dari usia, penghasilan, hingga tingkat pendidikan. Tanpa pemahaman data yang mendalam, strategi pemasaran berisiko tidak tepat sasaran dan gagal memaksimalkan potensi revenue.

Proyek ini dilakukan dalam rangka Studi Independen di PT Vinix Seven Aurum, dengan tujuan mengungkap pola tersembunyi dari dataset 388 pelanggan melalui EDA, uji statistik, dan data storytelling — menghasilkan 3 insight yang langsung dapat diterjemahkan menjadi keputusan bisnis strategis.

eda_visualisasi.py — Insight 1: Distribusi Usia
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv('online_food_clean.csv')
age_counts = df['Age'].value_counts().sort_index()
colors = ['#b8860b' if 23 <= x <= 26 else '#1a6b5e'
          for x in age_counts.index]

plt.figure(figsize=(10, 6))
plt.barh(age_counts.index, age_counts.values, color=colors)
plt.title('Mayoritas Pengguna Berusia 23–26 Tahun')
plt.gca().invert_yaxis()
plt.show()

Bagian 4

Uji Statistik

3 uji statistik dijalankan untuk membuktikan insight secara matematis: Welch's t-Test (usia vs feedback), Korelasi Pearson (usia vs penghasilan), dan Regresi Sederhana (penghasilan → usia).

uji_statistik.py
from scipy import stats
import pandas as pd
import numpy as np

df = pd.read_csv('online_food_clean.csv')

# Welch's t-Test: Usia vs Feedback
pos = df[df['Feedback'] == 1]['Age']
neg = df[df['Feedback'] == 0]['Age']
t, p = stats.ttest_ind(pos, neg, equal_var=False)
print(f"t={t:.3f}, p={p:.4f}")   # t=-3.608, p=0.0005

# Korelasi Pearson: Usia vs Penghasilan
income_map = {'No Income':0, 'Below Rs.10000':5000,
              '10001 to 25000':17500, '25001 to 50000':37500,
              'More than 50000':75000}
df['Income_Num'] = df['Monthly Income'].map(income_map)
r, p2 = stats.pearsonr(df['Age'], df['Income_Num'])
print(f"r={r:.4f}, p={p2:.2e}")   # r=0.5622, p=1.03e-33

p-value = 1.03e-33

Hubungan sedang-kuat yang sangat signifikan — semakin tua, cenderung semakin tinggi penghasilan.

Penghasilan menjelaskan 31.6% variasi usia. Setiap kenaikan satu tingkat penghasilan, usia rata-rata naik beberapa tahun.

Keterbatasan & Rekomendasi Data

⚠ Keterbatasan Data Saat Ini

  • Tidak ada data frekuensi pemesanan per pelanggan

  • Tidak ada data waktu atau lama pengiriman

  • Kolom income masih berupa kategori, bukan angka pasti

✓ Saran Variabel Tambahan

  • Frekuensi order per bulan (loyalitas pelanggan)
  • Rating aplikasi skala 1–5 per transaksi
  • Waktu pengiriman dalam menit
  • Jarak pengiriman dalam kilometer
  • Jenis makanan yang dipesan (kategori)
