Alur Proyek

Dari Data Mentah ke Keputusan Bisnis

Tugas 2
Pencarian & Perolehan Data
Menemukan dataset Online Food Business (388 baris, 6 kolom: Age, Monthly Income, Educational Qualifications, Latitude, Longitude, Feedback) yang relevan untuk analisis perilaku pelanggan platform makanan online.
Tugas 3
Data Cleaning
Pengecekan missing value, duplikat, outlier dengan metode IQR, inkonsistensi teks, dan konversi tipe data. Kolom Feedback dikonversi ke biner (Positive=1, Negative=0). Dataset dinyatakan bersih dan siap analisis.
Tugas 4
Exploratory Data Analysis (EDA)
Statistik deskriptif, visualisasi distribusi tiap variabel, dan analisis hubungan antar variabel. Membandingkan pendekatan grafik yang tepat vs kurang tepat untuk setiap insight.
Tugas 5
Data Storytelling
Membuat ulang 3 grafik utama dengan judul yang langsung berisi insight, warna strategis, dan layout bersih agar dapat dipahami oleh siapapun tanpa latar belakang teknis.
Tugas 6
Uji Statistik
Menjalankan 3 uji statistik: Welch's t-Test (usia vs feedback), Korelasi Pearson (usia vs penghasilan), dan Regresi Sederhana (penghasilan → usia). Membuktikan insight secara matematis.
Tugas 10
Presentasi & Infografis
Menyajikan 3 insight dalam slide deck lengkap (latar belakang → rekomendasi → timeline → budget → risiko) dan 1 infografis satu halaman dengan potensi kenaikan revenue 20–30% dalam 6 bulan.

Temuan Utama

3 Insight dari Data

Insight 01

Mayoritas Pelanggan Berusia 23–26 Tahun

70%+ pelanggan adalah kelompok usia dewasa muda dengan puncak di usia 23–24 tahun. Kelompok usia <20 dan >30 sangat jarang, menunjukkan risiko churn tinggi jika promosi tidak cocok.

Keputusan Bisnis Luncurkan kampanye digital targeted di Instagram & TikTok. Budget iklan Rp 50 juta, target engagement +30%, retensi +20%.
Insight 02

Penghasilan Tidak Memengaruhi Kepuasan Pelanggan

Distribusi feedback positif dan negatif merata di semua kategori income. Kepuasan lebih dipengaruhi kualitas layanan pengiriman dan UI aplikasi, bukan harga atau diskon.

Keputusan Bisnis Investasi Rp 100 juta ke peningkatan kualitas pengiriman dan UI. Target NPS +15 poin, churn −25%, save revenue Rp 2 M/tahun.
Insight 03

Pendidikan Tinggi Berbanding Lurus dengan Penghasilan

Post Graduate rata-rata berpenghasilan >70k, sementara School/Uneducated <40k. Promo generik berisiko tidak tepat sasaran dan melewatkan segmen premium.

Keputusan Bisnis Segmentasi promo: paket premium untuk Post Graduate, diskon untuk School/Uneducated. Target konversi +25%, revenue +10%.

data_cleaning.py Python 
import pandas as pd
import numpy as np
import tkinter as tk
from tkinter import filedialog

# Upload file CSV lewat dialog
root = tk.Tk()
root.withdraw()
file_path = filedialog.askopenfilename(
    title="Pilih file CSV dataset Online Food",
    filetypes=[("CSV files", "*.csv")]
)
df = pd.read_csv(file_path)

# Step 1: Profiling awal
print(df.info())
print(df.describe())

# Step 2: Missing value
print(df.isnull().sum())

# Step 3: Duplikat
print(df.duplicated().sum())

# Step 4: Outlier (IQR method)
for col in df.select_dtypes(include=[np.number]).columns:
    Q1 = df[col].quantile(0.25)
    Q3 = df[col].quantile(0.75)
    IQR = Q3 - Q1
    outlier = df[(df[col] < Q1 - 1.5*IQR) | (df[col] > Q3 + 1.5*IQR)]
    print(f"Outlier '{col}': {len(outlier)}")

# Step 5: Inkonsistensi teks
for col in df.select_dtypes(include=['object']).columns:
    print(df[col].value_counts())

# Step 6: Konversi tipe data
df['Feedback'] = df['Feedback'].replace({'Positive': 1, 'Negative': 0})

# Step 7: Simpan data bersih
df.to_csv('online_food_clean.csv', index=False)
eda_visualisasi.py Python 
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# Load dataset bersih
df = pd.read_csv('online_food_clean.csv')

# --- Insight 1: Distribusi Usia ---
age_counts = df['Age'].value_counts().sort_index()
colors = ['#b8860b' if 23 <= x <= 26 else '#1a6b5e'
          for x in age_counts.index]

plt.figure(figsize=(10, 6))
plt.barh(age_counts.index, age_counts.values, color=colors)
plt.title('Mayoritas Pengguna Berusia 23–26 Tahun')
plt.gca().invert_yaxis()
plt.show()

# --- Insight 2: Penghasilan vs Feedback ---
income_map = {'No Income': 0, 'Below Rs.10000': 5000,
              '10001 to 25000': 17500, '25001 to 50000': 37500,
              'More than 50000': 75000}
df['Income_Num'] = df['Monthly Income'].map(income_map)

sns.boxplot(data=df, x='Feedback', y='Income_Num', palette='muted')
plt.title('Kepuasan Tidak Dipengaruhi Penghasilan')
plt.show()

# --- Insight 3: Pendidikan vs Penghasilan ---
edu_order = (df.groupby('Educational Qualifications')['Income_Num']
               .median().sort_values().index.tolist())
sns.boxplot(data=df, x='Educational Qualifications',
            y='Income_Num', order=edu_order, palette='pastel')
plt.title('Pendidikan Memengaruhi Variasi Penghasilan')
plt.xticks(rotation=45)
plt.show()

Keterbatasan & Rekomendasi Data

⚠ Keterbatasan Data Saat Ini

  • Tidak ada data frekuensi pemesanan per pelanggan
  • Tidak ada rating per transaksi (hanya Positive/Negative)
  • Tidak ada data waktu atau lama pengiriman
  • Tidak ada jenis/kategori makanan yang dipesan
  • Kolom income masih berupa kategori, bukan angka pasti

✓ Saran Variabel Tambahan

  • Frekuensi order per bulan (loyalitas pelanggan)
  • Rating aplikasi skala 1–5 per transaksi
  • Waktu pengiriman dalam menit
  • Jarak pengiriman dalam kilometer
  • Jenis makanan yang dipesan (kategori)
