Latar Belakang & Masalah

Di era digital, konsumen memiliki preferensi yang kompleks sehingga strategi pemasaran tidak bisa lagi bersifat umum. Perusahaan perlu memahami karakteristik pelanggan secara mendalam untuk menyusun strategi yang tepat sasaran.

Proyek ini menjawab dua pertanyaan utama: Bagaimana menerapkan K-Means Clustering untuk mengelompokkan pelanggan berdasarkan usia, pendapatan tahunan, dan skor pengeluaran? Dan berapa jumlah cluster optimal serta apa karakteristik masing-masing cluster?

Dataset yang digunakan adalah Mall Customers Dataset dari Kaggle — 200 entri dengan atribut CustomerID, Gender, Age, Annual Income (k$), dan Spending Score (1–100).

Metodologi

Pipeline Analisis

1

Preprocessing & Standarisasi

Menghapus kolom non-numerik (CustomerID, Gender), lalu menstandarisasi Age, Annual Income, dan Spending Score menggunakan Z-score agar skala setiap fitur seragam dan tidak mendominasi perhitungan jarak.

StandardScaler
2

Menentukan Jumlah Cluster Optimal

Menggunakan Elbow Method dengan menghitung WCSS (Within-Cluster Sum of Squares) untuk k=2 hingga k=10. Titik siku teridentifikasi di k=4 — penambahan cluster setelah ini tidak memberikan penurunan WCSS yang signifikan.

Elbow Method WCSS
3

Clustering dengan K-Means++

Menjalankan K-Means dengan inisialisasi K-Means++ untuk centroid awal yang lebih optimal, menghitung Euclidean Distance tiap data ke centroid, dan mengiterasi hingga pengelompokan stabil (tidak ada perpindahan data antar cluster).

KMeans++ Euclidean Distance
4

Interpretasi & Rekomendasi

Menghitung rata-rata fitur per cluster, mengklasifikasikan karakteristik tiap cluster (muda/dewasa/tua, pendapatan tinggi/menengah/rendah), dan menyusun rekomendasi strategi pemasaran yang actionable.

Implementasi

Cuplikan Kode Python

Program dibangun dengan 8 step modular menggunakan Python di VS Code, mulai dari input data via dialog GUI hingga interpretasi otomatis hasil clustering.

# Step 6: Menentukan Jumlah Cluster Optimal
from sklearn.cluster import KMeans
import numpy as np

K = range(2, 11)
inertia = []

for k in K:
    kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
    kmeans.fit(X_scaled)
    inertia.append(kmeans.inertia_)

# Deteksi elbow point otomatis
diff  = np.diff(inertia)
diff2 = np.diff(diff)
k_optimal = K[np.argmax(diff2) + 1]
# → k_optimal = 4
python · clustering.py 
# Step 7: K-Means Clustering dengan inisialisasi K-Means++
kmeans = KMeans(
    n_clusters=4,
    init='k-means++',
    random_state=42,
    n_init=10
)
kmeans.fit(X_scaled)
labels = kmeans.labels_

# Tambahkan label ke dataframe asli
data['Cluster'] = labels

# Rata-rata per cluster untuk interpretasi
cluster_summary = data.groupby('Cluster')[fitur_terpilih].mean()
print(cluster_summary)
